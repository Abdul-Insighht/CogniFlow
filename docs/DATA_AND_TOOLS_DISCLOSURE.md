# Data And Tools Disclosure

## Data Sources

CogniFlow uses only:

- User-entered notes.
- User-entered tasks.
- User-entered natural language check-ins.
- A synthetic sample scenario written for demonstration.

CogniFlow does not use:

- Scraped websites.
- Proprietary data.
- Third-party personal data.
- Medical records.
- Student records from any institution.
- External APIs.

## Synthetic Data Method

The built-in sample scenario was manually written to represent a common student workload situation:

- One urgent quiz.
- One assignment.
- One lower-priority task.
- A short stressed check-in.
- Two short study notes.

No real person's private data was used.

## Software Tools

- HTML for structure.
- CSS for layout and interface styling.
- JavaScript for local state, cognitive-state scoring, note retrieval, and planning.
- Browser localStorage for persistence.
- Browser Web Crypto API for local IDs.
- AI coding assistant for code and documentation support.

## External Libraries

None.
